# frontend-design skill

## Purpose

This skill provides guidance and ready-to-use snippets for frontend design tasks: component structure, responsive layouts, accessibility considerations, and example HTML/CSS/React snippets.

## How to use

- Provide a short `task` describing the UI you'd like (e.g. "responsive hero with CTA").
- Optionally specify `locale` for language-specific labels (e.g. `de` or `en`).

## Outputs

- A markdown design proposal.
- Example code snippets (HTML/CSS/React) tailored to the request.
- Accessibility and responsive recommendations.

## Notes

Adapt file names and metadata to match your runtime/skill-loader conventions.
